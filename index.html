Projects

  • Computational Neuroscience

    In my computational neuroscience course, I developed and analyzed models of neurons like the leaky integrate-and-fire and exponential LIF neurons. I also looked at neural populations and different ways to encode information using spikes, like time-to-first-spike. Two important concepts we studied were synaptic plasticity rules that allow neurons to learn - STDP and reward-modulated STDP. I implemented these models and learning rules in Python and analyzed how neuronal circuits could encode information and adapt based on experience. All my course projects and results are available in my GitHub repository if you want to see more details!

  • Inhibitory Synaptic Plasticity

    In this project, I explored inhibitory synaptic plasticity using computational models of neural networks. I implemented a learning rule called inhibitory STDP (iSTDP) in Pytorch, which models how inhibitory synapses change based on the timing of pre- and postsynaptic spikes. When I tested it in balanced network models, I found that iSTDP led to less activation and more stable firing rates in the excitatory neurons. I also looked at iSTDP in feedforward inhibition networks, and saw that it significantly reduced the target neuron's firing rate. Additionally, I combined iSTDP with reinforcement learning algorithms, and found that distinct groups of neurons emerged that specialized for different input patterns.

  • Predict Phone Number Price

    I did an end-to-end data science project where I tried to predict the prices of phone numbers. I scraped phone number listings from different websites and cleaned up the data to create a dataset. I did some exploratory data analysis to understand what attributes seemed related to the price. Then I trained some machine learning models, like random forests and convolutional neural networks in PyTorch, to predict the price based on the phone number attributes. Next, I tuned the hyperparameters. Finally, I evaluated the models using metrics like MSE to compare their performance.

  • Computer Vision

    This project involves computer vision core techniques in Python—topics like image classification, convolutional neural networks, and different image filters. I built image classifiers using transfer learning and fine-tuning in TensorFlow. I also implemented the ResNet50 architecture in PyTorch to get better at CNN fundamentals. In addition, I created convolutional and pooling layers and DoG and Gabor filters for low-level image processing tasks. All the code and documentation are in a repository, showing the details if you're interested!

  • Machine Learning Algorithms

    I implemented important machine learning algorithms from scratch in Python as part of a deep learning course. The algorithms I coded up include K-Nearest Neighbors, Linear Regression, Logistic Regression, and Multi-Layer Perceptrons. For linear regression, I implemented both the normal equation and gradient descent methods. I also coded logistic regression two ways - in NumPy and in PyTorch using auto-grad for the optimization.

  • Connect-4

    The Connect-4 game project exemplifies my skills in game AI and adversarial search algorithms. I built a fully playable version of the classic connection game in Python using PyGame for graphics. The project showcases Minimax search with depth-limited lookahead to implement an unbeatable AI opponent. The evaluation function includes heuristics for assessing the strength of board positions and identifying winning moves. To improve the user experience, I added colorful graphics using PyGame. The well-documented codebase follows OOP principles with classes for game state, visualization, and game logic.
